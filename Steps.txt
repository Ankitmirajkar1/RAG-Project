1. Created a folder
2. Open VS code
3. Create keys for openAI_api and Langchain_api
4. created .env file in the project folder in vs code
	- we created variables for openai_api key and langchain_api_key
5. created the requirements.txt
	- mention all the dependencies to be installed in your project environment
6. Create the virtual environment - conda create -p <<your_virtual_env_name>> python==3.10 -y
7. Activate the virtual environment - conda activate /<<your_virtual_env_name>>
8. Create .ipynb file with proper name. this will consist of your project code
9. If streamlit app is created, you need a seperate app.py file to run it - streamlit run app.py
